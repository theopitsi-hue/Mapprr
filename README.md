# Mappr
A fantasy map editor, created in Unity 2022.3.21f1.

### Planned features:
- [ ] Voronoi-based world map
- [ ] Smart loading-unloading of areas for performance
- [ ] Texture Map support for ground and ocean layers
- [ ] Zoom in to city view to be able to add details to roads and houses
- [ ] Infinite pan while in city view
- [ ] Sprite Decals/Stickers with grouping, layer and zoom visibility assignments
- [ ] Text labels with custom fonts, curvature, sizes and visibility ranges.
- [ ] Export entire world map or screen-visible city to PNG
